ELECTRIC VEHICLES

We investigated Electric Vehicles to determine whether or not we could predict the price of an electric vehicle based on certain features.

We created a two-step prediction model using Machine Learning Classification techniques of XG Boost Classifiers for the model.

We created visualizations of our dataset that allows for greater exploration of sales, top makers and models as well as top miles per hour, Front or Rear Wheel Drive, etc. We also included the data tables we used for our prediction model and visualizations with interactive features on our Data page.

The Status Predictor page allows you to predict the price of an electric vehicle by selecting correlating features. Similarly, the Analysis page provides greater detail and insight into the process we took of importing, cleaning, and feature engineering our data set in order to create our Machine Learning Model using XG-Boost Classifiers.
